This is the TransferItemApp API.¶

class app.api.waiter.views.TransferItemApp¶

This View is used to transfer “orders” or “items” to a table or an existing order. SCENARIO 1.

IF an item is to be a (gift) from table_1 to table_2. then a new order id will be created and a comment inserted. table_1 will be charged with the gift and the order will appear only on table one.

SCENARIO 2.

if table_1 wants to (treat) table_2 an already existing item (SERVED PREVIOUSLY on table_2) that item will be moved from table_2 to table_1 and a new order will be createD (on table_1).
